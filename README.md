## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

## Instala la cli de nest
```
npm i -g @nest/cli
```

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

# Ejecturar Docker

```bash
# levantar base de datos

docker-compose up -d
```

# Recontruir la base de datis con la semilla
```
http://localhost:3000/api/v1/seed
```
